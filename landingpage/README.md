# QCU Land Landing Page

This is a promotional website for the QCU Land project.

## How to Run

1.  **Navigate to the `landingpage` directory:**
    ```bash
    cd landingpage
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm start
    ```

This will open the website in your default browser.
